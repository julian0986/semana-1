Comandos Aprendidos en la terminal.

  1. ls: Muestra lo que hay en un directorio
  2. pwd: Muestra el directorio actual.
  3. cd: Cambia de directorio
  4. cat: Muesta lo que tiene el archivo.
  5. mkdir: se usa para crear carpetas
  6. touch: se usa para crear archivos

Creación de Alias

La creación y ultilización de los alias son muy importantes y nos ayuda a la optimización del tiempo cuando estamos trabajando con linux.

  • alias cl="clear"
  • alias l="ls"
